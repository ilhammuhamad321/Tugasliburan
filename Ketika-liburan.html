Hari-1

Liburan pada Hari pertama saya pada Pagi hari Saya membeli Model kit lagi karena saya sedikit bosan dan juga saya butuh hiburan agar saya merasa bahagia ataupun senang, Saya membeli model kit ini pada sekitar jam 09:00 AM
Pada jam 13:00 PM Saya hanya membaca alquran sekitaran 1 Juz dan seingat saya saya membaca sampai jam 14:00 PM dan setelah itu saya istirahat dan siap siap untuk sholat ashar dan sehabis itu saya langsung  menyiapkan hal hal untuk berbuka

Hari-2

Di hari kedua saya hanya bermain game dengan teman teman saya di discord yaitu bermain game Roblox dan juga CSGO. Saya dan juga teman teman saya bermain game tersebut dari pagi hingga sore sekitaran jam 16:50 PM, namun tidak melupakan sholat kami karena itu adalah hal wajib
ketika abis sholat maghrib dan juga berbuka saya hanya membaca alquran 1 juz lagi hingga isya sambil menunggu adzan isya dan saya pun melakukan ibadah solat isya dan melakukan solat tarawih hingga jam 20:00 PM

Hari-3

Pada hari ke tiga saya akhirnya merakit model kit yang saya beli pada hari pertama liburan dan saya merakit itu sekitar 2-3 jam an dari jam 09:12 AM dengan perasaan senang sekaligus perasaan agak gak enak aja, saya merakit ini dirumah saya
Pada siang hari saya bermain Minecraft dengan 3 teman saya untuk menuju ke The-end dan build build di overworld, Saya bermain dengan mereka untuk menghilangkan rasa bosan saya yang tidak melakukan apa apa pada waktu itu, jadi saya memilih ikut join bersama mereka di server milik teman saya

Di jam Sekitar Jam 13 an Saya membuka paket saya yang dari tokopedia Yang ditunggu sekitar 4 jam an dan saya rakit hingga sekitaran jam 17:00

Hari-7

Hari-8

Di jam 06:00 AM Saya merakit Model kit yang saya beli kemarin dan saya merakit Kit ini sekitaran 5-7 Jam karena selain partnya kecil kecil, jari jari saya sudah berlemak semua 
Pada siang harinya saya pergi makan bersama orang tua saya di sebuah restoran, Untuk lokasi dan nama saya sebenarnya gaktau jadi mohon maaf kalau lokasi dan nama restorannya gak ada, tapi untuk makanannya enak sekali sampe nambah 2 porsi

Hari-9

Pada jam 5 setelah sholat subuh saya membaca alquran lagi dan membaca Juz 30 sampai juz 29 sampai jam 6 an dan setelah itu saya memakan sarapan saya di lantai 1
Pada jam 7 an saya hanya menonton Review model kit Premium dan saya mengulang terus hingga 3 kali dan saya kembali bermain PS4 Saya di lantai pertama hingga maghrib namun saya gak melupakan ibadah saya sendiri

Hari-10

Di pagi hari jam 7 setelah saya sarapan saya langsung merakit Model kit yang kemarin saya beli dengan waktu 2 jam an dan sehabis itu saya langsung Main laptop saya dan bermain game Minecraft
Setelah saya merakit kit itu saya langsung mabar dengan beberapa teman discord saya untuk bermain game seperti CSGO, Valorant, Minecraft,Roblox, Dan juga battlefield 1 sampai sekitaran jam 5 sorean, dan setelah itu saya menyudahi bermain dan mandi serta siap siap sholat

Hari-11

Pagi harinya saya hanya menikmati pagi hingga siang dengan bermain game saja karena saya tidak tau harus apa di hari itu dan juga Keluarga lagi pergi ke rumah keluarga Paman kedua
Setelah saya selesai bermain game saya langsung memakan Risol yang udh di siapin dan saya makan dengan lahap semua risolnya dan rasa risolnya yang enak dan cukup membuat saya kenyang

Hari-12

Pada jam 8 pagi tiba tiba teman saya mengetok pintu rumah saya kalau ada hadiah dari teman lama saya yang udh tinggal di sumatra dan mengirimkan hadiah kit sebagai THR untuk saya, saya berterimakasih kepadanya karena memberikan hadiah 
Saya akhirnya merakit kit ini dari pagi hingga malam karena sambil melakukan ibadah, makan, dan juga mandi dan pada akhirnya di jam 7 malam saya berhasil merakit Kit yang diberikan kepada saya oleh teman saya yang berada di Sumatra, saya langsung merakit kit ini sebagai tanda terimakasih karena telah memberikan kit ini kepada diri saya

Hari-13

Saya dan beberapa teman saya akhirnya memainkan game World of warship pada jam 8 pagi setelah saya sarapan dengan roti telur, Saya ikut bermain karena saya bosan dirumah gak ada siapa siapa jadi saya memilih meluangkan waktu untuk bermain dengan teman teman saya 
Tiba tiba Ada paket yang datang ketika saya bermain game World of war ship di jam 3 siang dan ternyata paket ini adalah paket yang diberikan oleh teman Facebook saya di Komunitas Gundam dan saya berterimakasih kepadanya

Hari-14

Saya dan teman saya memainkan world of warship lagi dan saya memainkan Kapal carrier yaitu kapal yang membawa pesawat untuk menyerang kapal musuh, saya tidak menyukai itu karena tidaklah gaya saya dan sangatlah susah memakai kapal carrier, Jadi ya teman teman saya termasuk saya kalah terus dalam permainan
Karena saya sedikit emosi karena kalah melulu dan menunggu keluarga saya kembali lagi saya akhirnya membaca alquran kembali dari Juz 28 Hingga Juz 23 Untuk mengisi waktu saya

Hari-15

Pada pagi hari jam 9 ketika saya udh selesai sarapan saya hanya menonton Video review Transformers lagi, dan lagi, dan lagi karena saya sangatlah bosan dan gaktau harus melakukan apa hingga jam 12 siang
